About me:

I'm a Digital Marketing student at Bellevue College, and I recently added Web Development to my skillset. I did a little bit of coding in HTML and CSS in an intro to computer science class in high school, so I went into BATECH 189 with some experience. But since that was a full decade ago, I thought that it would be a good time to familiarize myself with the entire process again, and learn about new concepts I had no experience with before.

The most important concepts I learned and demonstrated:

  • HTML basics
  • Semantic HTML
  • Floating HTML elements
  • CSS basics
  • CSS Box Model
  • Creating repositories
  • Using Google Web Fonts
  • Embedding Youtube Videos
  • Adhering to typography rules
  • Validating code
